# Saketano
Rails練習用お酒の感想登録アプリ

### Saketanoとは
* * *
お酒を検索して、評価、写真を登録することができます。

### URL
* * *
https://saketano.herokuapp.com/

### セットアップ時の注意点
* * *
rake db:migrate
後、
rake db:seed
を実施してください。

実施しないと、お酒のデータが無いため、お酒を検索できません。

### 機能一覧
* * *

1. 認証
 - 会員登録
 - ログイン
 - ログアウト
2. マイページ
 - 自分の感想の表示
3. トップページ
 - みんなの感想の表示
4. お酒の検索及び感想登録
 - お酒の検索（インクリメンタルサーチ）
 - 感想、写真登録
